Tour de l’Abitibi Glencore 2023

23 Application GPS FollowMee

Nous utilisons le site FollowMee afin de pouvoir localiser:

  • le devant de la course : voiture du directeur technique;
  • le peloton : voiture du président du jury;
  • la queue de la course : le camion balai.

Pour ce faire, les personnes concernées doivent utiliser l’application sur leur téléphone cellulaire.

Application à installer


Il est possible de télécharger gratuitement la version correspondante à votre cellulaire grâce au code QR:

Paramètres de l’application

Dans menu / settings :

  • username : tour2023
  • password : communiquer avec Bruno Gauthier
  • Device name : Direction Course / Peloton / Balai

Pour assurer le bon fonctionnement :

  • Accepter que l’application vous suive en tout temps.
  • Presser sur start avant la course et stop après.
  • Fermer l’application entre les courses pour éviter de drainer la batterie de votre cellulaire.

Visualisation de la carte

Afin de visualiser, l’équipe de production TV peut accéder au site FollowMee.

Une fois connecté avec l’identifiant, il est possible d’aller sur l’onglet Map et de créer un lien pour le suivi.
